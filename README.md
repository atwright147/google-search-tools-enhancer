# Google Search Tools Enhancer Chrome Extension

This is a reference app for best practices regarding React, Redux, Webpack and their associated tools.

## Dev build for Chrome

Point Chrome at the `build` folder

```sh
npm install
npm run dev
```

## Production build

```sh
npm install
npm run build
```
