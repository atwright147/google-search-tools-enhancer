# React Template

This is a reference app for best practices regarding React, Redux, Webpack and their associated tools.

## Installation

Clone this repo and run:

```sh
npm install
```

## Try it out

```sh
npm run dev:stubs
```

## Production build

The following command will create a full build into the `dist` folder.

```sh
npm run build
```
